Python Getting Started

Download Python and follow the instructions from the official website.

Create a "helloworld.py" file with code:

print("Hello, World!")

In the terminal, go to the file's location and run command:

python helloworld.py

The output should be "Hello, World!" In the future, you can run any Python file with this command and the file name.

Comments

Start comments with "#", and Python will ignore the part of the line after it. Example:

#A comment

With a pair of triple-quotes, multiple lines between them will be ignored. Example:

"""
Comment line 1
Comment line 2
"""
Variables

To create a variable, simply assign a value to it, there is no need to declare its type. Example:

a = 2
b = "aaa"
print(a)
print(b)
Data Types

By default, the built-in data types are:

  • Text Type: str
  • Numeric Types: int, float, complex
  • Sequence Types: list, tuple, range
  • Mapping Type: dict
  • Set Types: set, frozenset
  • Boolean Type: bool
  • Binary Types: bytes, bytearray, memoryview
  • None Type: NoneType
Functions

Define a function with "def", and call it with its name. Example:

def function_1():
print("This is function 1.")

function_1()

To pass arguments, specify them when defining and pass them when calling. Multiple arguments should be seperated by commas. Example:

def function_add(a, b):
print(a + b)

function_add(1, 3)
